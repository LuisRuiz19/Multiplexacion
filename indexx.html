Tipos de Multiplexación

Representación visual simplificada de cómo se transmiten los datos en un medio compartido.

1. TDM (División de Tiempo)

Los usuarios se turnan. El Usuario 1 pasa, luego el 2, luego el 3, y se repite.

Usuario 1
Usuario 2
Usuario 3
Usuario 1
Usuario 2

Nota: Todo el ancho de banda se usa por breves instantes de tiempo.

2. FDM (División de Frecuencia)

Cada usuario tiene su propio "carril" o frecuencia. Todos viajan juntos al mismo tiempo sin chocarse.

Nota: Se requieren bandas de guarda (espacios) entre las ondas para evitar interferencia.

3. WDM (División de Longitud de Onda)

Exclusivo de Fibra Óptica. Múltiples colores de luz viajan por el mismo hilo de vidrio.

Nota: Cada color lleva datos diferentes a velocidades extremas.

4. CDM (División de Código)

Todos transmiten a la vez en la misma frecuencia, pero con un "código" o patrón matemático único.

SEÑAL CODIFICADA MEZCLADA

Nota: Solo quien tiene el código puede separar la señal azul de la roja dentro de la mezcla.
